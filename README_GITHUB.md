# RBRLIGHT_PASSAPORTO_LUCI

Pacchetto finale demo per GitHub Pages.

## Pubblicazione
1. Crea repository `RBRLIGHT_PASSAPORTO_LUCI`.
2. Carica tutti i file e cartelle contenuti in questo ZIP nella root del repository.
3. Vai su `Settings > Pages`.
4. Seleziona `Deploy from a branch`, branch `main`, folder `/root`.
5. Link atteso: https://lightrbr.github.io/RBRLIGHT_PASSAPORTO_LUCI/

## Contenuti
- Google Maps integrata tramite iframe, senza API key.
- 10 tappe geolocalizzate.
- Immagini reali delle installazioni.
- Timbri digitali salvati nel browser.
- Attestato finale generato come PNG.
- QR demo e QR per singola tappa.
- Locandina dimostrativa.

## Nota
Questa versione è dimostrativa e potrà essere personalizzata secondo grafica ufficiale, contenuti definitivi e indicazioni della Stazione Appaltante.
